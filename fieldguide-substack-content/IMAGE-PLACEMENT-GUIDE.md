## Image Naming Convention

Save all images in: `fieldguide-substack-content/images/`

### Naming pattern:
- Book covers: `[book-name]-cover.jpg`
- Practice screenshots: `practice-[##]-[name].jpg`
- Principle inspiration: `[principle-name]-inspiration.jpg`
- General: `[descriptive-name].jpg`

### Image specs:
- Format: JPG or PNG
- Max width: 1200px
- Optimize for web (compress)
- Maintain aspect ratio
- Keep file sizes under 500KB
