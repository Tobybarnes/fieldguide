## Beginning

### Start a web app
```prompt
Build me a [type] web app that [does what]. Keep it simple—just HTML, CSS, and JavaScript in a single file.
```

### Create a landing page
```prompt
Create a landing page for [product/project]. Include a headline, description, and call-to-action button.
```

### Basic CRUD app
```prompt
Build me a simple app where I can create, read, update, and delete [items]. Store the data in localStorage.
```

### Form with validation
```prompt
Create a form that collects [fields]. Validate the inputs and show helpful error messages.
```

### Data display
```prompt
Build a page that displays this data: [paste data]. Make it searchable and sortable.
```

## Expanding

### Add search
```prompt
Add search functionality to this code. It should filter results as I type.
```

### Add filtering
```prompt
Add the ability to filter [items] by [criteria]. Let me select multiple filters.
```

### Add sorting
```prompt
Add sorting options: [list options]. Remember my preference.
```

### Add local storage
```prompt
Make this data persist using localStorage. It should survive page refreshes.
```

### Add export
```prompt
Add a button to export the data as [JSON/CSV]. Download it as a file.
```

## Fixing

### Debug error
```prompt
I'm getting this error: [paste error]. Here's my code: [paste code]. What's wrong and how do I fix it?
```

### Not working as expected
```prompt
This code should [expected behavior] but instead it [actual behavior]. Here's the code: [paste]
```

### Styling issue
```prompt
The layout is broken. [Describe issue]. Here's my CSS: [paste]. How do I fix it?
```

### Performance issue
```prompt
This is running slowly when [condition]. How can I make it faster?
```

### Mobile not working
```prompt
This works on desktop but not on mobile. Here's the code: [paste]. What needs to change?
```

## Polishing

### Improve design
```prompt
Make this look more polished. Current code: [paste]. Keep functionality the same but improve the visual design.
```

### Add dark mode
```prompt
Add a dark mode toggle to this. It should remember the preference.
```

### Make responsive
```prompt
Make this responsive. It should work well on mobile, tablet, and desktop.
```

### Add animations
```prompt
Add subtle animations to make interactions feel smoother. Keep it tasteful.
```

### Improve typography
```prompt
Improve the typography. Better font choices, sizes, line heights, and spacing.
```

## Releasing

### Prepare for deploy
```prompt
I want to deploy this to Vercel. What changes do I need to make to the code?
```

### Add meta tags
```prompt
Add proper meta tags for SEO and social sharing. Title: [title], Description: [desc]
```

### Optimize images
```prompt
How should I optimize images for web? What formats and sizes?
```

### Add favicon
```prompt
How do I add a favicon to my site?
```

### Custom domain
```prompt
I have a custom domain. How do I connect it to my Vercel deployment?
```

## Maintaining

### Add feature
```prompt
Add [feature] to this existing code: [paste]. Keep the current functionality working.
```

### Refactor
```prompt
Refactor this code to be cleaner and more maintainable: [paste]
```

### Update styling
```prompt
Change the color scheme to [colors] while keeping the layout the same.
```

### Fix after update
```prompt
This stopped working after [change]. How do I fix it?
```

### Add backup
```prompt
Add a way to backup all my data and restore from a backup file.
```
