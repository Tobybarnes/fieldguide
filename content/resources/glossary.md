---
title: Glossary
description: Plain-language definitions for terms you'll encounter.
---

## API
Application Programming Interface. A way for programs to talk to each other. When you fetch weather data, you're using an API. [Learn more at MDN](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Client-side_web_APIs/Introduction)

## Browser
The program you use to view websites (Chrome, Safari, Firefox). It reads HTML, CSS, and JavaScript and shows you the result.

## CSS
Cascading Style Sheets. The code that makes things look good—colors, fonts, layouts, spacing. [CSS-Tricks](https://css-tricks.com) has excellent guides.

## Deploy
Making your thing live on the internet so others (or you from other devices) can access it.

## Function
A reusable chunk of code that does something specific. Like a recipe you can use over and over.

## Git
A system for tracking changes to code. Like version history in Google Docs, but for code. [GitHub's guide](https://docs.github.com/en/get-started/using-git/about-git) explains the basics.

## HTML
HyperText Markup Language. The structure of a webpage—headings, paragraphs, links, images. [MDN's HTML guide](https://developer.mozilla.org/en-US/docs/Learn/HTML) is the definitive reference.

## JavaScript
The programming language that makes websites interactive. Clicks, animations, data fetching—that's JavaScript. [JavaScript.info](https://javascript.info) is a clear, modern tutorial.

## JSON
JavaScript Object Notation. A way of structuring data that's easy for both humans and machines to read.

## LocalStorage
A way to save data in the browser that persists even when you close the tab. Good for personal tools.

## MCP
Model Context Protocol. A way to connect Claude to external data like your calendar, email, or files. See [Using MCPs](/resources/mcps) for setup guides.

## NPM
Node Package Manager. A way to download and use code other people have written. [npmjs.com](https://www.npmjs.com) hosts millions of packages.

## Repository
A folder containing your project and its history. Often hosted on [GitHub](https://github.com).

## Terminal
A text-based way to interact with your computer. You type commands, it does things.

## Variable
A name that holds a value. Like a labeled box that stores something.

## Vercel
A platform for deploying websites. Connect your code, click deploy, get a URL. [vercel.com](https://vercel.com) offers a generous free tier.
